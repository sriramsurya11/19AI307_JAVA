## Sourcecode.java:
```
import java.util.*;
public class StringBufferExample3{  
    public static void main(String args[]){ 
        Scanner sc=new Scanner(System.in);
        String str1=sc.nextLine();
        StringBuffer sb=new StringBuffer(str1);  
        sb.replace(1,3,"Java");  
        System.out.println(sb); 
}  
}
```
